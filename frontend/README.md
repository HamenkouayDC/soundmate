# SoundMate Frontend

Frontend-часть проекта SoundMate — приложения для знакомств на основе музыкальных вкусов.

## Стек

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router

## Требования

Перед запуском нужно установить:

- Node.js LTS
- npm
- запущенный backend SoundMate

Backend по умолчанию должен работать по адресу:
